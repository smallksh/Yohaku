# 余白 Yohaku

> 浏览之后，不必留下所有痕迹。

余白是一个浏览器扩展。它不提供传统的无痕模式，而是允许你在正常浏览的同时，决定哪些记录应该留下，哪些记录应该成为余白。

## ✨ 功能特性

- **留白会话**：一键开始/结束，期间正常浏览
- **记录选择**：结束留白后，可以精确选择要删除/导出的记录
- **三种导出格式**：JSON / CSV / Markdown
- **深色模式**：浅色 / 深色 / 跟随系统
- **状态持久化**：浏览器重启后自动恢复会话状态
